# Data Directory Guide

This folder holds raw/input data only. Anything generated by a script (compiled
tables, figures) belongs in `results/`, not here.

## Core tables (start here)

| File | Description |
|---|---|
| `site_id_convention.md` | Naming rules for site IDs (`[STREAM]-[POSITION]`), the full stream-code table, and the Beaver Creek name-collision disambiguation. Read this before adding any new site. |
| `master_site_lookup.csv` | One row per sampling site: `site_id`, `stream_code`, `stream_position`, `site_name`, `waterbody_name`, `lat_dd`/`long_dd`, `date_established`, `N_samples_to_date`, `notes`. The reference table everything else joins against. |
| `sample_log_water.csv` | One row per water sample collected: `sample_id`, `site_id`, `collection_date`, `collection_notes`. Joins to `master_site_lookup.csv` on `site_id`. |
| `water_sample_summary.xlsx` | Manually maintained tally of sample counts per stream per year (`N_2024`/`N_2025`/`N_2026`). Used as a cross-check/QA reference in `R/01_compile_sr_data.R` — if the compiled counts don't match this, something's out of sync. |

## Subfolders

| Folder | Description |
|---|---|
| `icpms/` | Raw ⁸⁷Sr/⁸⁶Sr ICP-MS results, one workbook per analysis batch (`Samples` + `QAQC` sheets). File names indicate sample year + analyst: `sr_water_2024_rjohnson.xlsx`, `sr_water_2025_bmaitland.xlsx`. Joined to the sample log via `sample_id`. |
| `otoliths/` | Otolith/tissue sample inventories from IDFG and RMRS — not yet incorporated into the Sr compilation; reference material for future otolith microchemistry work. |
| `literature/` | Supporting data extracted from published studies (Bataille 2020, Bourret et al. 2022, Hamann 2012 appendices). Not MFSR-specific field data — used for cross-region comparison and lit-review context. |
| `spatial/` | GIS layers. `hu8_mfsr*.gpkg` = HUC8 basin boundary; `flowline_pn17_nsi_mfsr.gpkg`/`mfsr_main.gpkg` = stream network; `IsoWorld.gdb` = digitized Sr water/otolith sample points from prior published MFSR-area studies (Hamann 2012, Heckle 2019, Hegg & Kennedy 2013, Bourret 2023, Pearcy 2018, Lowe 2020, Ciepiela); `geology/idaho_lithology.shp` = USGS State Geologic Map Compilation for Idaho (bedrock lithology polygons, WGS84). |

## Provenance notes

- All file/folder names use `snake_case`; this was a full rename pass (2026-07) from
  the original inconsistent names (spaces, typos, mixed case) — see git history for
  the old→new mapping if you need to trace a file back to an email attachment or
  original source.
- `IsoWorld.gdb` is kept as a single ESRI File Geodatabase bundle rather than split
  into individual files — renaming risks breaking internal ESRI metadata, so it's
  documented here instead.
